什么是 DOM？

DOM (Document Object Model) 是一种用于表示和操作 HTML、XML 文档的编程接口。它将文档解析为一个由节点和对象组成的树结构，使开发者能够通过编程语言（如 JavaScript）来访问和修改文档的内容、结构和样式。

DOM 提供了一种结构化的方式来表示文档，使得开发者可以：

  • 访问和修改文档中的元素
  • 添加新元素到文档中
  • 删除文档中的元素
  • 修改元素的属性和样式
  • 响应用户交互事件

DOM 树结构

DOM 将文档表示为一个树状结构，其中每个节点都是文档的一部分。树的顶部是根节点（通常是 <html> 元素），然后分支到子节点，形成一个层次结构。

DOM 树示例：

└── html
    ├── head
    │   ├── title
    │   └── meta
    └── body
        ├── header
        │   └── h1
        ├── nav
        │   └── ul
        │       └── li
        ├── section
        │   ├── h2
        │   └── p
        └── footer
            └── p

在 DOM 树中，每个节点都可以是以下类型之一：

  • 元素节点：如 <div>、<p>、<h1> 等
  • 文本节点：元素内的文本内容
  • 属性节点：元素的属性，如 id、class、src 等
  • 注释节点：文档中的注释
  • 文档节点：整个文档的根节点

常见 DOM 元素

HTML 文档由各种元素组成，每种元素都有其特定的用途和属性。以下是一些常见的 DOM 元素：

块级元素

块级元素会独占一行，通常用于组织页面结构。

  • <div> - 通用块容器
  • <p> - 段落
  • <h1> 至 <h6> - 标题
  • <section> - 文档分区
  • <article> - 独立内容
  • <header> - 头部区域
  • <footer> - 底部区域

内联元素

内联元素不会独占一行，通常用于文本中的内容。

  • <span> - 通用内联容器
  • <a> - 超链接
  • <strong> - 强调文本
  • <em> - 斜体文本
  • <img> - 图像
  • <br> - 换行
  • <input> - 输入控件

表单元素

用于收集用户输入的元素。

  • <form> - 表单容器
  • <input> - 输入字段
  • <textarea> - 文本区域
  • <select> - 下拉选择框
  • <button> - 按钮
  • <label> - 表单标签

DOM 操作方法

JavaScript 提供了多种方法来操作 DOM 元素。以下是一些常用的 DOM 操作方法：

方法 描述
getElementById() 通过 ID 获取元素
getElementsByClassName() 通过类名获取元素集合
getElementsByTagName() 通过标签名获取元素集合
querySelector() 通过 CSS 选择器获取第一个匹配的元素
querySelectorAll() 通过 CSS 选择器获取所有匹配的元素
createElement() 创建新元素
appendChild() 向元素添加子元素
removeChild() 从元素中移除子元素
setAttribute() 设置元素属性
getAttribute() 获取元素属性
innerHTML 设置或获取元素的 HTML 内容
textContent 设置或获取元素的文本内容

DOM 操作实例演示

示例 1：修改元素内容

点击按钮修改下方段落的内容：

这是初始文本内容
function changeText() { const element = document.getElementById('output1'); element.textContent = '文本内容已被修改！'; }

点击按钮在下方添加新的段落：

function createElement() { const newParagraph = document.createElement('p'); newParagraph.textContent = '这是新创建的段落元素'; const container = document.getElementById('output2'); container.appendChild(newParagraph); }

示例 3：修改元素样式

点击按钮修改下方元素的样式：

这是一个可以修改样式的元素
function changeStyle() { const element = document.getElementById('output3'); element.style.backgroundColor = '#e67e22'; element.style.color = 'white'; element.style.padding = '15px'; element.style.borderRadius = '5px'; }

DOM 事件处理

DOM 允许我们为元素添加事件监听器，以响应用户的交互行为。常见的事件包括：

鼠标事件

  • click - 点击事件
  • mouseover - 鼠标悬停
  • mouseout - 鼠标离开
  • mousedown - 鼠标按下
  • mouseup - 鼠标释放

键盘事件

  • keydown - 按键按下
  • keyup - 按键释放
  • keypress - 按键按下并释放

表单事件

  • submit - 表单提交
  • change - 元素值改变
  • input - 输入框输入
  • focus - 元素获得焦点
  • blur - 元素失去焦点

事件监听器示例

点击下方按钮触发事件：

// 使用 addEventListener 添加事件监听器 document.getElementById('eventBtn').addEventListener('click', function() { document.getElementById('eventOutput').textContent = '按钮被点击了！'; });

DOM 遍历

DOM 提供了多种方法来遍历节点树，包括：

属性 描述
parentNode 获取元素的父节点
childNodes 获取元素的所有子节点
firstChild 获取元素的第一个子节点
lastChild 获取元素的最后一个子节点
nextSibling 获取元素的下一个兄弟节点
previousSibling 获取元素的上一个兄弟节点
